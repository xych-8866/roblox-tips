How to Prepare for Events

  • Save bucks: Event pets cost 5k-10k each. Save up before the event starts.
  • Save trade items: Keep valuable pets and potions – you'll need them for trading.
  • Collect event currency daily: Most events have daily quests that give extra currency.
  • Check the event board: Visit the event area daily for updates and quests.
